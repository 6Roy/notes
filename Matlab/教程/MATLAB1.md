# &gt; MATLAB操作界面

---

* **操作面板**  
	1. 文件目录（文件结构的显示）
  	2. 编辑器（打开的脚本文件）
  	3. 命令行窗口（用于随时操作数据）
  	4. 工作空间（用于储存变量的空间）
  	5. 命令历史记录（用于储存命令行中的数据）
  	6. 导航栏（主页、绘图、应用程序、编辑器、发布、视图）

> 使用tab键调节嵌套中的缩进问题。  


* **辅助命令**  
	1. path（搜索文件的目录）
	2. help path（查看帮助）
	3. cd（current directory）
	4. userpath（用户使用的路径）
	5. savepath（保存路径）
